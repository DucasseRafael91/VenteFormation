# Application de Vente de Formations

## Description du projet
Cette application permet aux utilisateurs de consulter et d’acheter des formations en ligne. Elle est conçue pour offrir une expérience simple et intuitive, que ce soit pour naviguer parmi les formations ou pour passer commande.

## Fonctionnalités

### Phase 1 – Consultation des formations
- Affichage de toutes les formations disponibles.
- Filtrage par catégorie.
- Recherche par mot-clé dans le nom ou la description.
- Filtrage selon le mode de formation : présentiel ou distanciel.
- Affichage des détails de chaque formation : nom, description, durée, prix, mode (présentiel/distanciel).

### Phase 2 – Gestion du panier et commandes
- Possibilité d’ajouter ou de retirer des formations dans un panier.
- Passage de commande à tout moment, uniquement si l’utilisateur est connecté.
- Si l’utilisateur n’est pas connecté, redirection vers la création de compte (login + mot de passe).
- Chaque commande est associée à un client (nom, prénom, email, adresse, téléphone).
- Un utilisateur peut réaliser plusieurs commandes pour différents clients.

## Objectif
- Permettre aux utilisateurs de consulter facilement les formations.
- Faciliter le processus d’achat et de gestion des commandes.
- Offrir une expérience utilisateur fluide, que ce soit pour les visiteurs ou les clients enregistrés.
